# 《寒蝉鸣泣之时·奉/粹》简体中文翻译项目

[English](README.md)

这里是《寒蝉鸣泣之时·奉/粹》的简体中文翻译项目。
汉化文本基于官中以及社区补丁翻译。

## 如何打包

### 奉（NS）

1. 创建 `raw/hou` 文件夹。

2. 将 2.0.2 版本的游戏文件放入该文件夹，包括 `data.rom`、`patch.rom` 和 `append.rom`。

3. 在终端中运行 `sh pack.sh hou`。

### 粹（PSV）

1. 创建 `raw/sui` 文件夹。

2. 将 `data.rom` 放入该文件夹。

3. 在终端中运行 `sh pack.sh sui`。
