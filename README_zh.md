# 《寒蝉鸣泣之时·奉》简体中文翻译项目

[English](README.md)

## 快速开始

### 如何打包

1. 创建一个 `raw` 文件夹，并将 2.0.2 版本的游戏文件放入其中，包括 `data.rom`、`patch.rom` 和 `append.rom`。

2. 在终端中运行 `sh pack.sh`。
