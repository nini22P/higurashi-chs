# Higurashi Hou Chinese Translation Project

[中文文档](README_zh.md)

## Quick start

### How to pack

1. Create a `raw` folder and place the game files for version 2.0.2 inside it, including `data.rom`, `patch.rom`, and `append.rom`.

2. Run `sh pack.sh` in the terminal.
